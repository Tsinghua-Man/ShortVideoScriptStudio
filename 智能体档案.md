- 编号：A001

- 当前目录：`C:\Users\Administrator\Desktop\智能体集合\已落地智能体\短视频脚本生成智能体`

## 作用定位

用于根据结构化 brief 自动生成一版可直接改写、可直接拍摄参考的短视频脚本草案，也支持输出“系统提示词 + 结构化需求”的提示词包。

## 当前包含内容

- `agent/`
  - 系统提示词、输入模板、输出规范、评估清单
- `examples/`
  - 示例 brief
- `src/`
  - Python 生成器与本地 Web 服务
- `web/`
  - 可直接打开的前端页面
- `start_agent.bat`
  - 命令行一键启动
- `start_web_ui.bat`
  - 网页版一键启动
- `USAGE_CN.md`
  - 中文使用说明

## 启动入口

- 命令行版：`start_agent.bat`
- 网页版：`start_web_ui.bat`
- 直接浏览器打开：`web/index.html`

## 当前能力

- 根据平台、目标、脚本类型、时长生成结构化短视频脚本
- 自动补充钩子、标题建议、分镜建议、字幕重点、CTA
- 输入不完整时，自动补默认假设，避免直接卡死
- 支持 `口播`、`带货`、`知识分享`、`测评`、`剧情`、`探店`
